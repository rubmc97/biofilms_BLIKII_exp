# Agricultural land use shapes short and long‑term bacterial diversity, community structure, and assembly in biofilms of adjacent streams
Rubén Martínez-Cuesta12*; Rebecca Hoess3; Sebastian Floßmann4; Juergen Geist3; Michael Dannenmann4; Michael Schloter12; Stefanie Schulz2

The repository containst the following files:
manuscript_biofilm_script.R: R Script for the 16S rRNA processing from sequencing to phyloseq object + further analyses (alpha, beta div, ANCOMBC2, co-occurrence networks...)
iCAMP_script_biofilms.R: iCAMP script
metadata_blik_2_exp_0424.csv: 16S rRNA amplicon sequencing metadata deployed to obtain the results described in the manuscript importing it into manuscript_biofilm_script.R
